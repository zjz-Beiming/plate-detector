# 🚗 YOLOv8 车牌检测器

基于 YOLOv8 的中国车牌实时检测桌面应用，支持图片、视频、摄像头和网络流输入，采用 PyQt6 构建暗色主题界面。

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![PyQt6](https://img.shields.io/badge/PyQt6-6.5+-green.svg)
![YOLOv8](https://img.shields.io/badge/YOLOv8-Ultralytics-yellow.svg)

## ✨ 功能特性

- **4 种输入模式** — 图片文件 / 本地视频 / 摄像头实时流 / RTSP/HTTP 网络流
- **实时检测** — QThread 异步检测线程，UI 不卡顿
- **左右双栏布局** — 左侧显示原图+标注框，右侧展示裁剪车牌缩略图
- **车牌颜色分类** — 自动识别蓝牌（民用车）、黄牌（大型车辆）、绿牌（新能源）
- **暗色主题** — 现代暗色 UI 设计
- **保存结果** — 保存带检测框标注的结果图片
- **视频控制** — 播放/暂停/停止 + 进度条拖动定位
- **网络流重连** — 断线自动重连（最多 3 次）
- **可配置设置** — 切换模型权重、调整置信度阈值

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Windows / macOS / Linux
- 摄像头（可选，用于实时检测）

### 安装

```bash
git clone https://github.com/zjz-Beiming/plate-detector.git
cd plate-detector
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```

首次运行会自动下载车牌检测模型，需联网。

## 📖 使用说明

### 图片检测
1. 点击 「📁 打开图片」 选择图片文件
2. 自动检测并显示标注结果
3. 右侧面板展示裁剪出的车牌区域
4. 点击 「💾 保存标注图」 保存结果

### 视频检测
1. 点击 「🎬 打开视频」 选择视频文件
2. 自动逐帧检测，支持播放/暂停/停止
3. 拖动进度条可跳转到指定位置

### 摄像头检测
1. 点击 「📷 打开摄像头」 输入设备索引（默认 0）
2. 实时检测画面中的车牌

### 网络流检测
1. 点击 「🌐 网络流」 输入 RTSP 或 HTTP 地址
2. 自动检测并显示连接状态
3. 断线自动重连（最多 3 次）

### 设置
- 点击 「⚙ 设置」 可切换模型权重文件或调整置信度阈值

## 🎨 车牌颜色分类

| 类型 | 颜色 | 说明 |
|------|------|------|
| 蓝牌 | 🔵 蓝底白字 | 小型民用车 |
| 黄牌 | 🟡 黄底黑字 | 大型车辆/教练车 |
| 绿牌 | 🟢 绿底白字 | 新能源车辆 |

分类基于 HSV 颜色空间分析裁剪区域的主色调。

## 🔧 技术架构

- **UI 层** — PyQt6 左右双栏布局 + 暗色主题样式表
- **检测引擎层** — Ultralytics YOLOv8 封装，支持模型热切换
- **数据流层** — OpenCV VideoCapture 统一处理所有视频源
- **线程安全** — QThread + 信号/槽机制，检测线程与 UI 线程完全隔离
